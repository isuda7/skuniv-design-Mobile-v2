Publishing Guide

Scrolling

Default

한 번의 스크롤 이벤트로 모든 상태를 구분/관리 한다.

스티키, 그림자, 올림, 내림, Top버튼 등을 관리 한다.


스크롤 구분 클래스는 body에 추가한다. (script)

스크롤 구분 클래스는 modal에 추가한다. (script)

페이지 구분 클래스는 wrapper에 추가한다. (html)


페이지 스크롤 참조 : /assets/js/ui.common.js

모달 스크롤 참조 : ScrollType08.html

스크롤 구분

상태 (페이지, 모달)

스크롤된 상태: is_scrolld

위치 (페이지)

스크롤 최상위: is_scroll_first

스크롤 마지막: is_scroll_last

방향 (페이지)

스크롤 올림: is_scroll_up

스크롤 내림: is_scroll_down

페이지 구분 (페이지)

Sticky 있을 시: has_sticky 추가

AppNav 있을 시: has_appnav 추가

Page Floating 있을 시: has_floating 추가

Page Expand 있을 시: has_expand 추가
